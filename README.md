# private-notes 私人注释
点一下star ✨,是对作者最大的支持
#### 介绍

 你还在为项目中不敢添加 "敏感注释"!<br>

 源码是只读文件不能添加注释而烦恼吗？<br>

 在任何你想加注释的地方 按下**Alt + Enter**鼠标移出点击即可保存<br>

 已有私人注释 按下**Alt + Enter**即可编辑和删除<br>
![示例1](./doc/show.gif)

#### 安装
打开IntelliJ IDEA -> plugin，搜索 Private Notes，安装重启即可
